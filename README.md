# THE LOOP SKILLS

אוסף הסקילים של THE LOOP — כלים מוכנים לקלוד קוד, לבעלות עסקים.

## איך מתקינים סקיל

נכנסות לעמוד THE LOOP SKILLS, לוחצות "העתק התקנה" על הסקיל שרוצות, ומדביקות את הפקודה בקלוד קוד. זהו.

לדוגמה, התקנת סקיל כתיבת מודעות:

```bash
curl -sL https://github.com/shira-smtng/the-loop-skills/archive/main.tar.gz | tar -xz -C /tmp && cp -r /tmp/the-loop-skills-main/skills/ad-copywriter ~/.claude/skills/ && rm -rf /tmp/the-loop-skills-main
```

באהבה, שירה ודידי
